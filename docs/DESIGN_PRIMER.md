# 放置裂隙 — 设计精华版

放置型ARPG。角色自动挂机打怪，玩家回来做三件事：筛选装备、调整Build、决定挂哪层。

## 1. 战斗

纯数值模拟，60秒时限。公式：

```
DPS = ATK × (1 + 主属性×1%) × 攻速 × [1 + 暴击率×(暴伤-1)] × 元素增伤
EHP = HP / [(1 - 护甲/(护甲+500)) × (1 - 闪避率)]
战力 = √(DPS × EHP × 10)
```

胜负：`玩家击杀时间 = 怪物HP/DPS` < 60s 且 < `怪物击杀时间 = 玩家HP/怪物DPS`。

挂机每1200ms一场战斗，死亡/背包满/手动停止时暂停。

上限：攻速5.0 | 暴击率75% | 闪避60%。

## 2. 角色（三职业）

| 职业 | 主属性 | 流派A | 流派B |
|------|--------|-------|-------|
| 战士 | 力量 | 狂暴(暴击) | 钢铁(坦克) |
| 游侠 | 敏捷 | 影舞(爆发) | 风行者(攻速) |
| 法师 | 智力 | 元素使(DOT) | 奥术(冰控) |

主属性加成：力量+攻击%，敏捷+暴击率%，智力+元素伤害%，每点1%。

训练：攻击(+2攻/级)、体魄(+18血/级)、防御(+3护甲/级)。200级上限，消耗=基础×1.25^等级，边际递减（每级收益减20%）。

切换职业：首次转生前可自由切换。

## 3. 装备（核心系统）

`装备 = 基础物品 + 品质 + 等级 + 基础属性 + 词缀(2-6条) + 强化等级`

部位×9：武器、副手、头盔、护甲、手套、鞋子、左戒、右戒、项链。

品质×5：普通(1词缀) → 魔法(2) → 稀有(3-4) → 传说(4-5+传奇特效) → 远古(5-6+远古特效)。

传说率 = 0.1% + ln(怪物等级)×0.25%，硬上限2%。装备等级=floor(怪物等级/2+10)。基础属性缩放=1.05^装备等级（指数）。

### 3.1 词缀×18

| 词缀 | 效果 | 核心部位 | 类型 |
|------|------|---------|------|
| 锋利 | +攻击 | 武器 | 攻击 |
| 残忍 | +暴击伤害 | 手套 | 攻击 |
| 鹰眼 | +暴击率 | 项链 | 攻击 |
| 疾风 | +攻速 | 手套 | 攻击 |
| 烈焰/冰霜/雷霆 | +元素伤害 | 武器、项链 | 元素 |
| 活力 | +生命 | 护甲 | 防御 |
| 守护 | +护甲 | 头盔 | 防御 |
| 灵巧 | +闪避 | 鞋子 | 防御 |
| 烈焰/冰霜/雷霆抗性 | +对应抗性 | 项链 | 抗性 |
| 贪婪 | +金币获取 | 戒指 | 功能 |
| 寻宝 | +魔法发现 | 戒指 | 功能 |
| 吸血 | 伤害转生命 | 武器 | 功能 |
| 迅捷 | +移速 | 鞋子 | 功能 |

部位专属：武器=吸血，项链=全元素抗性，副手=格挡率，鞋子=移速。

词缀协同：暴击率+暴击伤害同时存在时评分加权×1.2。

### 3.2 评分

5档模式：均衡(全属性均权) / 暴击(暴击权重×2) / 攻速(攻速权重×2) / 坚韧(EHP权重×2) / 主属性(主属性权重×3)。

评分 = Σ(属性值×权重) + Σ(词缀值×权重×传奇倍率)，品质加成=1+品质阶级×0.12。

切换评分模式联动推荐天赋分支和训练优先级。

### 3.3 强化

| 等级 | 成功率 | 失败惩罚 | 备注 |
|------|--------|---------|------|
| +1~+3 | 100% | 无 | |
| +4~+6 | 85% | 无 | |
| +7~+9 | 65% | 降1级 | |
| +10 | 45% | 不变(保护) | 关键里程碑 |
| +11+ | 30% | 破碎(装备消失) | 需保护卷轴 |

消耗=基础×1.5^等级。只提升基础属性，不影响词缀。安全强化石：+7~+9失败不降级的保护机制。

### 3.4 套装（第二阶段）

3套：狂暴者(暴击)、守护者(坦克)、风暴使者(攻速元素)。2/4/6件激活，属性加成+触发特效。

### 3.5 传奇×15（第二阶段）

每件有独特触发特效：雷霆之怒(弹射)、血饮(吸血)、时间扭曲(暴击加速)、末日使者(斩杀增伤)、钢铁壁垒(护盾)、烈焰行者(站火增伤)、幽灵步(闪避必暴)等。

## 4. 推层与怪物

**层数结构**：普通层自动挂机，每10层Boss手动挑战，首通后开放挂机。每25层"秘藏层"掉落+100%但怪物有随机词缀。

**怪物类型**：均衡(标准) / 高血(×1.5HP) / 高攻(×1.5ATK) / 奖励(+30%掉率)。

**推荐战力**：普通层=100×1.12^(层数-1)，Boss层×1.5。

**收益衰减**（平滑曲线）：
```
收益 = min(1.0, 战力/推荐战力)^1.5
战力≥100%推荐 → 100%
80-100% → 80%
60-80% → 50%
<60% → 20%
```

**怪物词缀×8**（精英怪/秘藏层）：荆棘(反弹10%)、狂暴(低血加速)、护盾(5秒吸1次)、分裂(死后2小怪)、吸血(20%回复)、毒素(DOT叠加)、坚固(护甲+50%)、迅捷(攻速+30%闪避+15%)。

**Boss机制**：第10层召唤小怪(考验持续DPS)、第20层阶段性减伤(考验节奏)、第30层高爆发(需药水)、第40层生命恢复(需快速击杀)。首通奖励：必掉稀有+、大量资源、解锁新区域。

## 5. 离线收益

离线后按推荐挂机层自动战斗。12小时软上限：0-12h=100%、12-18h=50%、18-24h=25%、24h+=10%。

离线报告：时长/击杀/金币/经验/装备(分品质)/收益倍率。背包满时暂停不丢失。

时间倒流检测：lastOnlineTimestamp + 最大离线上限24h。

## 6. 转生（第二阶段）

条件：通关50层或战力达标。

重置：层数/等级/训练/天赋/金币/强化石。

保留：装备模板/背包扩容/成就/付费。

收益："灵魂"永久货币 = 层数^1.5 × 战力因子 × 传奇因子。

灵魂用途：8项全局加成（伤害+5%/金币+10%/掉率+8%/攻速+3%/生命+5%/离线+10%/强化率+3%/背包+5格），每项可多次购买成本递增。

第N次转生后：all属性×(1+N×0.05)。

里程碑：1次(解锁远古) → 3次(解锁第2职业) → 5次(解锁第3职业) → 10次(远古掉率+1%) → 25次(洗词缀) → 50次(神话品质)。

## 7. 经济

| 货币 | 来源 | 用途 |
|------|------|------|
| 金币 | 挂机、分解、离线 | 训练(无限)、强化、洗词缀、天赋重置 |
| 强化石 | 分解普通/魔法、过滤转化 | 装备强化 |
| 远古精华(50层后) | 分解传说、Boss首通 | 传说洗词缀、远古升级 |
| 灵魂(转生) | 转生获得 | 永久全局加成 |

goldFind/magicFind上限50。训练无限等级确保金币始终有去处。强化石软上限999。

## 8. 天赋（第二阶段）

48节点网状树，3职业×2路径=6分支，3层×4列。

层级解锁（点满下层解锁上层），终极天赋二选一。

天赋×装备联动：如"精准弱点"在装备含鹰眼词缀时效果翻倍。

重置：首次免费，后续500/1000/2000/4000...递增。

## 9. 背包与UI

50格初始，金币/强化石扩容（+5格/次，成本递增）。

拾取过滤：最低品质/保留部位/必须词缀/自动转化。三档预设（宽松/魔法+/稀有+）。

批量：一键分解低品质、按评分模式一键标记、智能保留（每部位最多5件最高分）。

压力：剩余20%警告、10%危急、满暂停。

装备对比：并排展示、属性差值绿↑红↓、DPS/EHP变化量、一键替换。

一键"穿戴最优"：根据当前评分模式自动装备最高分。

Build快照：保存配置，切换后对比变化。

UI：三栏布局（左角色/中战斗/右背包），标签页导航。

## 10. 视觉反馈

品质色：普通白 → 魔法蓝 → 稀有黄 → 传说橙 → 远古红。

传说掉落：全屏橙色光柱 + 底部公告 + 号角声。
远古掉落：全屏红光 + 全服公告 + 史诗音效。
推层：屏幕中央 + 烟花粒子。
离线开箱：金属声 → 传说优先展示 → 逐件揭晓。
+10强化：金光全屏爆发。
