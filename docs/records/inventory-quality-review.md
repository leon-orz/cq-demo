# 背包整理阶段复盘

## 完成情况

本阶段完成了自动拾取过滤、分解保护、装备锁定、评分对比和更优高亮的最小可行版本。

## 已验证能力

- 按最低品质过滤掉落。
- 被过滤装备自动转化为强化石。
- 装备评分可计算并与已穿戴装备比较。
- 锁定装备不会被一键分解。
- 评分更优的低品质装备不会被一键分解。
- 过滤预设可以切换和重置。

## 实现取舍

- 分解确认使用原生确认框，后续需要正式弹窗。
- 过滤规则暂只在右侧面板展示，后续可拆成独立面板。
- 装备评分是通用权重，后续需要按流派和主属性调整。
- 离线奖励目前领取时才执行过滤，和离线容量截断顺序仍需优化。

## 风险

- 当前评分可能不符合所有构筑，尤其是暴击流和坦克流。
- 自动转化低品质装备会减少背包压力，但玩家可能需要更明确的转化日志和统计。
- 原生确认框不适合移动端体验。

## 下一步建议

1. 做正式装备对比浮窗，展示属性差值和替换后 DPS/EHP 变化。
2. 做正式分解确认弹窗，列出将分解和已保护的装备。
3. 增加装备排序：品质、评分、部位、是否更优。
4. 调整离线收益流程，让过滤规则先于背包容量截断执行。
