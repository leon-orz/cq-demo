# 背包整理阶段复盘

## 完成情况

本阶段完成了自动拾取过滤、分解保护、装备锁定、评分对比和更优高亮的最小可行版本。

## 已验证能力

- 按最低品质过滤掉落。
- 被过滤装备自动转化为强化石。
- 装备评分可计算并与已穿戴装备比较。
- 锁定装备不会被一键分解。
- 评分更优的低品质装备不会被一键分解。
- 过滤预设可以切换和重置。

## 实现取舍

- 本阶段最初使用原生确认框，后续已由“装备对比与分解确认”阶段替换为正式弹窗。
- 过滤规则暂只在右侧面板展示，后续可拆成独立面板。
- 装备评分是通用权重，后续需要按流派和主属性调整。
- 离线奖励目前领取时才执行过滤，和离线容量截断顺序仍需优化。

## 风险

- 当前评分可能不符合所有构筑，尤其是暴击流和坦克流。
- 自动转化低品质装备会减少背包压力，但玩家可能需要更明确的转化日志和统计。
- 离线收益先截断再领取过滤，会让部分可转化装备占用离线奖励空间。

## 下一步建议

1. 调整离线收益流程，让过滤规则先于背包容量截断执行。
2. 拆分右侧面板，避免拾取过滤、背包整理和资源统计挤在一个组件里。
3. 增加按构筑权重计算装备评分的能力。
4. 补充自动转化日志和离线回归后的整理摘要。
